    • T-SQL(Transact-SQL)

    T-SQL是在标准SQL语句的基础上增加了用于控制程序执行顺序的语句
    （一）数据定义语言
    CREATE/ALTER/DROP
    （二）数据操纵语言
    SELECT/INSERT/UPDATA/DELETE
    （三）数据控制语言
    CRANT/REVOKE/DENY

    常量

    一旦确定，值不可变

    变量（局部变量/全局变量）

    值可以改变

    🐸局部变量

    声明局部变量需要使用 DECLARE语句

    DECLARE
{
    {
         {@local_variable [as] data_type} -----@local_variabke变量名，必须以@开头 ，/// data_type 变量类型
          | [=value] -----为变量赋值，必须数据类型匹配
         } 
   |  {@cursor_variable_name cursor}   -----@cursor_variable_name游标变量名称
} [,...n]      ------对多个变量进行赋值
| {
     @tabel_variable_name [as] <table_type_definition>
} -----定义表类型变量的名称和数据类型，注意，声明表类型变量时必须时改语句中声明的唯一变量

    例如：
    声明变量和赋值变量

    declare @cirditid char(18)='0000000000000'  ----声明变量时赋初值
declare @id int,@name varchar(20),@sex char(2),@brithday datetime
set  @id=1001 -----set添加单个数据
select @name='小小'
select @sex='女',@brithday='1998/6/24' ------select添加多条数据
select @id '学号',@name '姓名',@sex '性别',@brithday '出生日期',@cirditid '身份证号' -----给每个变量取别名并显示

    🐸全局变量

    一般由系统自身提供，不允许用户自定义，使用@@定义

    运算符

    主要分为 赋值运算符/算术运算符/字符串连接运算符/比较运算符/逻辑运算符/一元运算符/位运算符

    🐟赋值运算符

    '='

    1.直接赋值
    2.表达式建立联系

    🐟算术运算符

    '+'  '-'  '*'  '/'  '%'（取模）

    🐟字符串连接运算符

    '+' 连接的两边必须是字符串类型

    🐟比较运算符

'!>'不大于  
'!<'不小于
