    🌸2.安装

    安装包有两种格式，分别为.zip和.msi，msi需要安装，常用的还是.zip
    以我自己下载的为例，解压到D盘，产生文件夹mysql-8.0.20-winx64
    文件夹内容如下（下载的文件目录最好都是英文的，如果是中文后面安装时在cmd下可能会乱码）

    在根目录下创建my.ini (mysql配置文件)

    文件内容如下（需要将basedir 与 datadir 路径改成mysql的解压路径 ）

    [mysql]

# 设置mysql客户端默认字符集
default-character-set=utf8 

[mysqld]

#设置3306端口
port = 3306 

#设置mysql的安装目录（需要根据自己的安装路径更改）
basedir=D:\mysql-8.0.20-winx64\mysql-8.0.20-winx64

#设置mysql数据库的数据的存放目录（需要根据自己的安装路径更改）
datadir=D:\mysql-8.0.20-winx64\mysql-8.0.20-winx64\data

#允许最大连接数
max_connections=200

#服务端使用的字符集默认为8比特编码的latin1字符集
character-set-server=utf8

#创建新表时将使用的默认存储引擎
default-storage-engine=INNODB

    🌸3.以管理员身份运行CMD

    进入mysql的根目录bin下("cd "进入指定文件夹)

    输入下面命令，回车，第一次输入会等很长一段时间，不要着急，等到出现下一行的时候在根目录下会自动生成一个data文件（很多教程里面会在前面自己建立一个data文件，后来会出现一些问题，不建议这样做）

    mysqld --initialize-insecure --user=mysql

    输入下面命令，回车，会提示安装成功

    mysqld install

    🌸5.登录

    服务启动成功之后，需要登录的时候输入命令（这个时候如果提示“mysql”不是不是内部或外部命令，也不是可运行的程序或批处理文件，就需要再到bin目录下）

    mysql -u root -p

    第一次登录没有密码，直接按回车过

    🌸6.退出登录

    输入exit，退出登录

    🌸7.设置登录密码

    设置登录密码，在bin目录下输入下面命令

    mysqladmin -u root -p password

    一开始的密码直接按回车，然后输入新密码，确认新密码

    🌿注：如果出现安装mysql时报错: 由于找不到vcruntime140_1.dll,无法继续执行代码

    出现这种情况是因为缺少了动态链接库(.ddl文件), 跟mysql无关, 需要下载一份（我这里下载的是64位版本的）

    vcruntime140_1.dll下载地址

    下载最新版本的，解压后将vcruntime140_1.dll复制到C:\Windows\System32即可

    （32位版本的vcruntime140_1.dll需要复制到C:\Windows\SysWOW64下）
